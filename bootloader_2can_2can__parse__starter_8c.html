◆ initCANFilter()

bool initCANFilter ( void )

Configure the CAN2 start bank. There are 28 total filter banks that are shared between CAN1 and CAN2. The CAN2SB field indicates where the split between CAN1 and 2 is. A value of 14 means that 0...13 are for CAN1 and 14...27 are for CAN2.

Make sure that all CAN filter configuration is done with the CAN1 peripheral. CAN2 does not have access to modify/view the filters.
