# Data Sheet Library

Irving Wang (irvingw@purdue.edu)

## About the microcontrollers
- [RM0090](STM32F407_RM0090.pdf) contains register and peripheral information for F4 series boards.
- [RM0440](STM32G4_RM0440.pdf) contains register and peripheral information for G4 series boards.
- [STM Cortex-M4](STM32_cortexM4_PM0214.pdf) programming manual for the Cortex-M4 core itself, which is used in both F4 and G4 series boards.
- [ARM Cortex-M4](ARM_cortexM4_TRM.pdf) contains highly detailed information about the Cortex-M4 core, If you're looking at this, you're probably too far gone..

## Drivers
- [AMK](AMK_datasheet.pdf) contains information about the inverters, wiring diagrams, software interfaces, etc. for the AMK series inverters.

## Hardware
- [TJA1057](TJA1057_CAN_transciever.pdf) pinout and electrical characteristics for the CAN transceiver.
