# Data Sheet Library

Irving Wang (irvingw@purdue.edu)

## About the microcontrollers
- [RM0090](datasheets/STM32F407_RM0090.pdf) contains register and peripheral information for F4 series boards.
- [RM0440](datasheets/STM32G4_RM0440.pdf) contains register and peripheral information for G4 series boards.
- [STM Cortex-M4](datasheets/STM32_cortexM4_PM0214.pdf) programming manual for the Cortex-M4 core itself, which is used in both F4 and G4 series boards.
- [ARM Cortex-M4](datasheets/ARM_cortexM4_TRM.pdf) contains highly detailed information about the Cortex-M4 core, If you're looking at this, you're probably too far gone..

## Components
- [AMK](datasheets/AMK_datasheet.pdf) contains information about the inverters, wiring diagrams, software interfaces, etc. for the AMK series inverters.
- [Elcon Charger](datasheets/Elcon_charger.pdf) CAN protocol and operating procedure for HV battery charging via the Elcon charger.

## Sensors
- [Bosch LWS](datasheets/Bosch_LWS.pdf) Bosch steering angle sensor: pinout, CAN protocol, and electrical specs.
- [DHAB S/134](datasheets/DHAB_S134.pdf) LEM dual-channel hall-effect current sensor used on the HV bus.
- [IZZE IMU V2](datasheets/IZZE_IMU_V2_Datasheet.pdf) IZZE Racing IMU V2: accelerometer/gyro specs and CAN message layout.
- [ZED-F9P Protocols](datasheets/ZED-F9P_protocols.pdf) u-blox ZED-F9P UBX/NMEA protocol reference.
- [ZED-F9P Moving Base](datasheets/ZED-F9P-MovingBase.pdf) application note for running the ZED-F9P RTK GPS in moving-base mode (DAQ heading/velocity).

## Hardware
- [TJA1057](datasheets/TJA1057_CAN_transciever.pdf) pinout and electrical characteristics for the CAN transceiver.
- [W5500](datasheets/W5500.pdf) WIZnet hardwired TCP/IP ethernet controller: register map and SPI interface (used by DAQ).
