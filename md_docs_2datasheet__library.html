About the microcontrollers

  • RM0090 contains register and peripheral information for F4 series boards.
  • RM0440 contains register and peripheral information for G4 series boards.
  • STM Cortex-M4 programming manual for the Cortex-M4 core itself, which is used in both F4 and G4 series boards.
  • ARM Cortex-M4 contains highly detailed information about the Cortex-M4 core, If you're looking at this, you're probably too far gone..

Components

  • AMK Datasheet contains information about the inverters, wiring diagrams, software interfaces, etc. for the AMK series inverters.
  • AMK Diagnostics contains detailed list of diagnostic messages reported by the inverters for debugging.
  • Elcon Charger CAN protocol and operating procedure for HV battery charging via the Elcon charger.

Sensors

  • Bosch LWS Bosch steering angle sensor: pinout, CAN protocol, and electrical specs.
  • DHAB S/134 LEM dual-channel hall-effect current sensor used on the HV bus.
  • IZZE IMU V2 IZZE Racing IMU V2: accelerometer/gyro specs and CAN message layout.
  • ZED-F9P Protocols u-blox ZED-F9P UBX/NMEA protocol reference.
  • ZED-F9P Moving Base application note for running the ZED-F9P RTK GPS in moving-base mode (DAQ heading/velocity).
