A_BOX

A_BOX (short for "accumulator box") is the BMS master. It owns battery telemetry, the BMS shutdown rail (BMS_SDC_CTRL), and the charger control state machine. It is a safety-critical board.

Notable Files

  • main.c / main.h: Peripheral init (FDCAN1/2, SPI1, ADC1+DMA, GPIO), task startup order, and the bms_task periodic that runs ADBMS updates and pack/thermal fault checks.
  • CMakeLists.txt: Build configuration for the A_BOX target.
  • adbms/README.md: Canonical ADBMS6380 driver docs (high-level state machine, balancing, command/PEC flow). g_bms is shared with bms_task and the charger FSM.
  • charging_fsm.c / charging_fsm.h: Charger control FSM (CHARGING_FSM_PERIOD_MS = 1000) that clamps charge requests, toggles g_bms.is_balancing_enabled, and publishes charging_fsm_internals.
  • telemetry.c / telemetry.h: Multi-rate CAN telemetry publishers:
    • 100 Hz: pack_stats, charging_telemetry, rotating cell_telemetry
    • 8 Hz: rotating thermistor_telemetry
    • 0.2 Hz: abox_version with GIT_HASH
  • thermistor.c / thermistor.h: Datasheet-derived lookup table and thermistor_R_to_T() helper used by the ADBMS driver.
  • Fault handling in bms_task (in main.c): raises IMD, BMS_DISCONNECTED, pack full/empty, cell under/overvoltage, and pack warm/cold/overtemp faults; BMS_DISCONNECTED also drives BMS_SDC_CTRL.
